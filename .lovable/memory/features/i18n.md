---
name: i18n system
description: Site-wide UI language selector for EN/ES/FR/PT/SV/NO — translates all labels AND scripture verses
type: feature
---
- LanguageProvider in src/i18n/LanguageContext.tsx wraps App
- UI translations in src/i18n/translations.ts (TranslationStrings interface)
- Globe dropdown (LanguageSelector) in header next to Bible List link
- Language persisted in localStorage as "ui-lang"
- Scripture verse translations in src/data/verseTranslations.ts (127 verses × 5 languages)
- When non-English: 3-column layout = [Translation | KJV | Application] (NRSV hidden)
- When English: 3-column layout = [KJV | NRSV | Application]
- Bible translations used (all public domain):
  - Spanish: Reina-Valera 1909 (exact church match)
  - French: Louis Segond 1910 (exact church match, public domain)
  - Portuguese: João Ferreira de Almeida (exact church match)
  - Swedish: Swedish Bible 1917 (older public domain version; church prefers Folkbibeln 2015 which is copyrighted)
  - Norwegian: Norwegian Bible 1906/1930 (older public domain version; church prefers Norsk Bibel 2007 which is copyrighted)
