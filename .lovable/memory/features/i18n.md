---
name: i18n system
description: Site-wide UI language selector for EN/ES/FR/PT/SV/NO — translates all labels, not scriptures
type: feature
---
- LanguageProvider in src/i18n/LanguageContext.tsx wraps App
- Translations in src/i18n/translations.ts (TranslationStrings interface)
- Globe dropdown (LanguageSelector) in header next to Bible List link
- Scripture text (KJV/NRSV) never changes — only UI labels/buttons translate
- Church-approved Bible versions noted: Reina-Valera 1909, Louis Segond 1910, Almeida 1914, Svenska Folkbibeln 2015, Norsk Bibel 2007
- Language persisted in localStorage as "ui-lang"
