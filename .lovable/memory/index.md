# Project Memory

## Core
Historical Bible map app. KJV/NRSV translations. Restored Gospel topics from Elder Fisher CSV.
Writers: Paul, Peter, John, James, Jude, Hebrews Author. Google Maps default tile.
Site title: "The New Testament World: Apostles, Epistles & Journeys"
i18n: Site-wide UI language selector (EN/ES/FR/PT/SV/NO). Scripture text stays KJV/NRSV always.

## Memories
- [Scripture topics](mem://features/topics) — Searchable topic tags from CSV: Godhead, Priesthood, Apostacy, etc.
- [Writers & cities](mem://features/writers) — John at Patmos, Peter at Rome, James/Jude at Jerusalem, Hebrews at Alexandria, Titus at Crete
- [i18n system](mem://features/i18n) — UI translations in src/i18n/, LanguageProvider wraps App, globe dropdown in header
